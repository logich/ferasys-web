## Template Creation Workflow

### Using Canva MCP

When Canva MCP is available, use these commands to create templates:

1. **Create new design**: "Create a business card design for Feral Systems using the brand colors #00d4ff and #a855f7"

2. **Apply brand kit**: "Apply the Feral Systems brand kit to this design"

3. **Add elements**: "Add the horizontal logo to the top of this letterhead template"

4. **Export**: "Export this design as PNG at 300 DPI for print"

### Manual Creation Steps

1. Create new design with correct dimensions
2. Apply gradient background
3. Add logo from brand kit
4. Add text using brand fonts
5. Apply brand colors
6. Add gradient effects to headings
7. Save as template
8. Share with team

## Quality Checklist

Before finalizing any template, verify:

- [ ] Correct dimensions for intended use
- [ ] Brand colors used accurately (hex codes match)
- [ ] Correct fonts applied (Lastica for headings, Anonymous Pro for body)
- [ ] Logo has proper clear space
- [ ] Text is readable (meets WCAG AA standards minimum)
- [ ] Gradient applied correctly (45Â° or specified angle)
- [ ] All elements aligned properly
- [ ] Bleed and safe zones respected (for print materials)
- [ ] File exported at correct resolution (300 DPI for print, 72 DPI for web)

---

## Export Settings

### For Web Graphics
- **Format**: PNG or WebP
- **Resolution**: 72 DPI
- **Size**: Exact pixel dimensions specified
- **Color Profile**: sRGB

### For Print Materials
- **Format**: PDF (print-ready)
- **Resolution**: 300 DPI
- **Bleed**: Include 0.125" bleed
- **Color Profile**: CMYK

### For Social Media
- **Format**: PNG or JPG
- **Resolution**: Platform-optimized
- **Compression**: High quality (90%+)
- **Color Profile**: sRGB
